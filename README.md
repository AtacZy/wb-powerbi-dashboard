# 📊 Динамический дашборд Power BI для маркетплейса

Дашборд для анализа продаж, юнит-экономики и рекламных кампаний селлера на Wildberries.

## 📋 Описание проекта

Дашборд создан для владельца бизнеса на Wildberries. Позволяет ежедневно отслеживать ключевые показатели, анализировать прибыльность товаров и оценивать эффективность рекламных каналов.

### Возможности

- Мониторинг выручки, прибыли, заказов и остатков
- Юнит-экономика по каждому артикулу
- Анализ источников трафика и эффективности рекламы (ROAS, DRR)
- Сравнение продаж в дни акций и обычные дни
- Воронка: заказы → выкупы → отмены

## 📊 Данные

Выгрузка из личного кабинета селлера Wildberries за июнь–июль 2023.

**Исходный файл:** `wb-data-test.xlsx`

**Обработанный файл:** `wb_data_normalized.xlsx` (нормализован до модели «Звезда»)

## 📐 Модель данных

Модель «Звезда» из 5 таблиц:

| Таблица | Тип | Описание |
|:--|:--|:--|
| dim_Товары | Справочник | Артикулы и бренды |
| dim_Календарь | Справочник | Даты, дни недели, недели, месяцы |
| fact_Продажи | Факты | Заказы, выкуп, цены, акции, корзина |
| fact_Финансы | Факты | Прибыль, себестоимость, логистика, штрафы |
| fact_Маркетинг | Факты | Источники трафика, бюджеты на рекламу |

## 📈 Структура дашборда

3 страницы со сквозными фильтрами (дата, артикул, день недели, акция):

| Страница | Содержание |
|:--|:--|
| 📊 Общие показатели | 7 KPI-карточек, динамика выручки и заказов, прибыль по артикулам, доли расходов |
| 📦 Товары | Юнит-экономика, доли корзины, продажи по дням недели, воронка заказы→выкупы→отмены |
| 📢 Маркетинг | 4 KPI, расходы по неделям, каналы трафика, эффективность рекламы |

## 🧮 Меры DAX

Более 30 мер, включая сложные:

| Мера | Описание | Сложность |
|:--|:--|:--|
| `Себестоимость` | Себестоимость выкупленных товаров | `SUMX` + `VAR` + `CALCULATE` |
| `Комиссия_Детально` | Комиссия WB с учётом штрафов | `SUMX` + `VAR` + `CALCULATE` |
| `Стоимость_Заказа` | Стоимость привлечения одного заказа | `SUMX` + `VAR` + `CALCULATE` |

Также: `ROAS`, `Доля рекламных расходов`, `Маржинальность`, `Процент выкупа`, `Прибыль на единицу`, `Отмены шт.`, `Заказы с рекламы`.

## 🚀 Запуск

1. Открыть `Dasboard_marketplate.pbix` в Power BI Desktop
2. При необходимости обновить путь к `wb_data_normalized.xlsx`

## 🛠 Используемые технологии

- **Power BI** — визуализация, DAX, модель данных
- **Excel** — источник данных
